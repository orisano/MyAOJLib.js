## Run on

### Browser and node-webkit

```js
<script src="lib/MyAOJLib.js"></script>
<script>
console.log( MyAOJLib() );
</script>
```

### WebWorkers

```js
importScripts("lib/MyAOJLib.js");

console.log( MyAOJLib() );
```

### Node.js

```js
require("lib/MyAOJLib.js");

console.log( MyAOJLib() );
```
